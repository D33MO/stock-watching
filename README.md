# 📊 股票监控 — A股实时行情悬浮窗

一个基于 **PyQt6** 的 Windows 桌面悬浮窗工具，实时监控 **A股** 和 **期货** 行情，每只品种一行，显示 **名称 · 现价 · 涨跌幅 · 迷你分时图** 等可选字段。


![界面预览](image.png) (v1.0.0样式)

---

## ✨ 功能总览

### 🪟 悬浮窗
- **无边框悬浮** — FramelessWindowHint，无标题栏、不在任务栏显示
- **置顶显示** — 一键切换置顶/取消置顶（标题栏图钉按钮）
- **自由拖拽** — 左键拖拽窗口任意位置，关闭后自动记忆位置
- **透明背景** — 支持窗口整体背景完全透明仅显示文字和图表
- **窗口位置记忆** — 关闭时自动保存屏幕坐标，下次启动恢复
- **深色主题** — 低暗色 UI，适合长时间挂屏

### 📈 实时行情
- **A 股实时行情** — 通过新浪财经 HTTP API 获取，免费无需 Key
- **期货实时行情** — 支持上期所、大商所、郑商所、中金所等主力合约（AKShare + 新浪双数据源）
- **可调刷新间隔** — 支持 3 秒 / 5 秒 / 10 秒 / 30 秒 / 60 秒
- **并行刷新** — 多品种数据后台线程并行请求，不卡 UI
- **涨跌颜色** — 涨红跌绿，一目了然

### 📉 迷你分时图
- **每只品种独立分时图** — 当日 1 分钟级走势，右侧显示
- **涨跌方向着色** — 涨势红色折线 + 红色填充区域，跌势绿色
- **昨收基准虚线** — 以昨收价（期货为昨结算价）为基准线，涨跌对称显示
- **非交易时段保留** — 非交易时段显示最后的分时数据

### 📋 可定制显示字段
自由勾选要显示的字段，支持拖拽排序（配置中调整）：

| 字段 | 说明 | 数据源 | 更新频率 |
|------|------|--------|----------|
| `price` | 现价 | 新浪财经 | 实时 |
| `change_pct` | 涨跌幅 | 新浪财经 | 实时 |
| `change` | 涨跌额 | 新浪财经 | 实时 |
| `volume` | 成交量（手） | 新浪财经 | 实时 |
| `turnover` | 成交额 | 新浪财经 | 实时 |
| `high` | 当日最高 | 新浪财经 | 实时 |
| `low` | 当日最低 | 新浪财经 | 实时 |
| `volume_ratio` | 量比 | 腾讯财经 | 约 3 分钟 |
| `turnover_rate` | 换手率 | 腾讯财经 | 约 3 分钟 |
| `open_interest` | 持仓量（期货） | AKShare | 实时 |
| `intraday` | 迷你分时图 | 新浪/AKShare | 约 60 秒 |

- 数值智能格式化：低于 1 元显示 4 位小数，低于 10 元显示 3 位
- 成交量/额自适应单位：手/万手/亿手，元/万/亿

### 🎨 品种颜色自定义
- **股票名称颜色** — 默认为浅灰 `#E0E0E0`，可自定义
- **期货名称颜色** — 默认为橙色 `#FFAA00`，可与股票区分
- **15 种预设色板** + 系统颜色选择器（完整取色器）
- 实时预览效果

### 🛠️ 品种管理
- **智能添加** — 输入股票代码（如 `600519`）或期货代码（如 `CU2609`），自动获取名称
- **类型选择** — 普通股票无需指定类型；期货添加时选择「期货」类型
- **列表排序** — 上移/下移调整品种显示顺序
- **右键快捷删除** — 在悬浮窗右键菜单直接删除品种
- **品种代码前缀自动识别** — 6 开头 → 沪市，其他 → 深市

### 🖥️ 系统托盘
- **关闭最小化到托盘** — 点击关闭按钮不退出，后台静默运行
- **托盘右键菜单** — 显示窗口 / 设置 / 退出
- **双击托盘图标** — 显示/隐藏窗口切换
- **托盘图标** — 使用应用 Logo 作为托盘图标

### 🔄 自动更新
- **GitHub 版本检测** — 启动时自动检查 GitHub Releases 最新版本
- **版本比较** — 语义化版本号对比（`v1.0.0` → `v1.1.0`）
- **一键跳转下载** — 发现新版本时显示提示按钮，点击跳转下载页

### 🚀 开机自启
- **Windows 注册表方式** — 通过 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
- **设置面板一键开关** — 勾选/取消即生效
- **自启动标记** — 以 `--autostart` 参数启动，可区分正常启动

### 📦 打包发布
- **PyInstaller 打包** — 一键打包为独立 EXE（`build.bat`）
- **GitHub Actions 自动发布** — 打 tag 自动构建并发布 Release
- **SVG 资源打包** — 图标等 SVG 文件自动打包进 exe
- **高 DPI 支持** — 适配高分屏显示

## ⚡ 快速开始

### 环境要求

- Windows 7+

## 🔧 配置文件

在 `config.json` 中可手动编辑配置：

```json
{
    "stocks": [
        { "code": "600519", "name": "贵州茅台", "type": "stock" },
        { "code": "CU2609", "name": "沪铜2609", "type": "futures" }
    ],
    "display_fields": ["price", "change_pct", "intraday"],
    "refresh_interval": 5,
    "window_x": 0,
    "window_y": 800,
    "opacity": 0.9,
    "auto_start": false,
    "always_on_top": true,
    "transparent_bg": false,
    "stock_name_color": "#E0E0E0",
    "futures_name_color": "#FFAA00"
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `stocks` | array | 监控的品种列表，`code` 为代码，`name` 为名称，`type` 为 `stock` 或 `futures` |
| `display_fields` | array | 显示字段列表，可选值见上表 |
| `refresh_interval` | int | 实时行情刷新间隔（秒） |
| `window_x` / `window_y` | int | 窗口屏幕坐标（自动记忆） |
| `opacity` | float | 窗口透明度 0.0 ~ 1.0 |
| `auto_start` | bool | 是否开机自启动 |
| `always_on_top` | bool | 是否窗口置顶 |
| `transparent_bg` | bool | 是否窗口背景透明（true 时仅显示文字和图表） |
| `stock_name_color` | str | 股票名称颜色（十六进制，如 `#E0E0E0`） |
| `futures_name_color` | str | 期货名称颜色（十六进制，如 `#FFAA00`） |

## 📌 期货支持说明

- **代码格式**：品种代码 + 年份后两位 + 月份，如 `CU2609`（沪铜 2026 年 9 月合约）
- **涨跌基准**：期货使用 **昨结算价** 计算涨跌幅（非昨收价）
- **数据源**：优先 AKShare 获取国内期货行情，新浪 hf_ API 作为回退
- **特有字段**：可显示 **持仓量**（手 / 万手）
- **名称自动识别**：内置 40+ 品种映射表（沪铜、螺纹钢、甲醇、原油、黄金、白银等）
- **分时数据**：支持多周期尝试（1/5/15/30/60 分钟）

## 🧱 技术栈

| 组件 | 用途 |
|------|------|
| **PyQt6** | 桌面 GUI 框架（无边框窗口、系统托盘、SVG 渲染） |
| **新浪财经 API** (`hq.sinajs.cn`) | A股实时行情 HTTP 接口（免费，无需 Key） |

| **ThreadPoolExecutor / QThread** | 后台并行数据请求，不阻塞 UI |
| **PyInstaller** | 打包为独立 EXE |
| **GitHub Actions** | CI/CD 自动打包发布 |

## ❓ 常见问题

**Q: 行情数据不刷新？**
A: 检查网络连接；新浪 API 在交易时段返回实时数据，非交易时段返回收盘数据。可在设置中降低刷新间隔。

**Q: 分时图显示为空或"加载中..."？**
A: 非交易时段分时接口不返回数据，属于正常现象。交易时段如果仍为空，检查网络或稍后重试。

**Q: 如何开机自启动？**
A: 在设置面板勾选「开机自启动」即可；也可将 EXE 快捷方式放入 `shell:startup` 文件夹。

**Q: 期货代码怎么填？**
A: 格式为品种代码 + 年份后两位 + 月份，例如 `CU2609`（沪铜 2026 年 9 月合约）。添加时需在类型中选择「期货」。

**Q: 提示有新版本怎么更新？**
A: 设置面板中会显示「🔔 发现新版本」提示，点击「去下载」按钮跳转到 GitHub Releases 页面，下载最新版 exe 覆盖即可。

**Q: 期货名称显示不对？**
A: 内置了 40+ 常用品种映射表，如 `CU` → 沪铜、`RB` → 螺纹钢。如未覆盖可在 [data/fetcher.py](data/fetcher.py) 的 `FUTURES_NAME_MAP` 中自行添加。

**Q: 窗口不透明或透明效果不对？**
A: 在设置的「窗口行为」中勾选/取消「窗口背景透明」即可切换；效果即时生效。

**Q: 如何修改股票的排列顺序？**
A: 在设置面板的品种列表中，选中品种后点击「上移」或「下移」按钮调整顺序。

**Q: 配置文件的路径在哪？**
A: `config.json` 位于程序运行目录。打包为 EXE 后与 exe 同目录。
